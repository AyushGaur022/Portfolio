# Portfolio - Animated Vite + React

This project is the animated component-based React conversion of your original HTML portfolio.

## Features added
- Framer Motion entrance & hover animations
- Scroll-reveal via IntersectionObserver
- Typing animation retained
- Smooth active-nav highlighting while scrolling
- Button glow & ripple effect
- Image zoom on hover for cube icons
- Dark/light theme with animated transitions

## How to run
npm install
npm run dev
